# Hızlı Başlangıç: Yeni Başlayanlar İçin Vektör Arama

Qdrant, veri kümenizdeki yaklaşık en yakın veri noktalarını bulmak için tasarlanmıştır. Bu hızlı başlangıç kılavuzunda, uzay kolonilerini izlemek için basit bir veritabanı oluşturacak ve vektör temsiline göre en yakın koloniyi arayacaksınız.

<Alert severity="info">API isteğini göndermek için **ÇALIŞTIR**'a tıklayın. Yanıt, kod parçacığının altında görünecektir.<br/>Ayrıca herhangi bir kod bloğunu düzenleyip isteği yeniden çalıştırarak farklı sonuçlar görebilirsiniz.</Alert>

## Adım 1: Koleksiyon oluşturun

İlk olarak, koloni verilerini depolamak için `star_charts` adında bir koleksiyon oluşturacağız. Her konum, dört boyutlu bir vektör ile temsil edilecek ve benzerlik araması için mesafe ölçütü olarak Dot product kullanacağız.

Koleksiyonu oluşturmak için bu komutu çalıştırın:

```json withRunButton=true
PUT collections/star_charts
{
  "vectors": {
    "size": 4,
    "distance": "Dot"
  }
}
```

## Adım 2: Koleksiyona veri yükleyin

Koleksiyon hazır olduğuna göre, şimdi biraz veri ekleyelim. Her konum bir vektöre ve adı gibi ek bilgilere (payload) sahip olacak.

Verileri eklemek için bu isteği çalıştırın:

```json withRunButton=true
PUT collections/star_charts/points
{
  "points": [
    {
      "id": 1,
      "vector": [0.05, 0.61, 0.76, 0.74],
      "payload": {
        "colony": "Mars"
      }
    },
    {
      "id": 2,
      "vector": [0.19, 0.81, 0.75, 0.11],
      "payload": {

      }
    },
    {

      "vector": [0.36, 0.55, 0.47, 0.94],
      "payload": {
```

## Adım 3: Arama sorgusu çalıştırın

Şimdi, belirli bir vektörü temsil eden uzamsal konuma en yakın üç koloniyi arayalım. Bu sorgu, kolonileri payload bilgileriyle birlikte döndürecektir.

En yakın kolonileri bulmak için aşağıdaki sorguyu çalıştırın:

```json withRunButton=true
POST collections/star_charts/points/search
{
  "vector": [0.2, 0.1, 0.9, 0.7],
  "limit": 3,
  "with_payload": true
}
```
