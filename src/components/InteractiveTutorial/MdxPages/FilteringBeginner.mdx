# Temel Filtreleme - Cümleler ve Koşullar

## Adım 1: Koleksiyon Oluşturun

İlk olarak, `terraforming` adında bir koleksiyon oluşturun. Her nokta, boyutu 4 olan vektörlere sahip olacak ve mesafe ölçütü `Dot` olarak ayarlanacak:

```json withRunButton="true"
PUT collections/terraforming
{
  "vectors": {
    "size": 4,
    "distance": "Dot"
  }
}
```

## Adım 2: Vektör ve Payload İçeren Noktalar Ekleyin

Şimdi koleksiyona noktalar ekleyin. Her nokta bir `id`, `vector` ve `land type`, `color`, `life presence` ve `humidity` gibi çeşitli niteliklere sahip bir `payload` içerir:

```json withRunButton="true"
PUT collections/terraforming/points
{
  "points": [
    {
      "id": 1,
      "vector": [0.1, 0.2, 0.3, 0.4],
      "payload": {"land": "forest", "color": "green", "life": true, "humidity": 40}
    },
    {
      "id": 2,
      "vector": [0.2, 0.3, 0.4, 0.5],
      "payload": {"land": "lake", "color": "blue", "life": true, "humidity": 100}
    },
    {
      "id": 3,
      "vector": [0.3, 0.4, 0.5, 0.6],
      "payload": {"land": "steppe", "color": "green", "life": false, "humidity": 25}
    },
    {
      "id": 4,
      "vector": [0.4, 0.5, 0.6, 0.7],
      "payload": {"land": "desert", "color": "red", "life": false, "humidity": 5}
    },
    {
      "id": 5,
      "vector": [0.5, 0.6, 0.7, 0.8],
      "payload": {"land": "marsh", "color": "black", "life": true, "humidity": 90}
    },
    {
      "id": 6,
      "vector": [0.6, 0.7, 0.8, 0.9],
      "payload": {"land": "cavern", "color": "black", "life": false, "humidity": 15}
    }
  ]
}
```

## Adım 3: Filtrelemeden önce alanları indeksleyin

**Not:** Filtreleme yapmadan önce bir alanı her zaman indekslemelisiniz. İndeks oluşturmadan önce filtreleme kullanırsanız, Qdrant tüm veri kümesinde yapılandırılmamış bir şekilde arama yapar. Arama performansınız çok yavaş olur.

```json withRunButton="true"
PUT /collections/terraforming/index
{
    "field_name": "life",
    "field_schema": "bool"
}
```

```json withRunButton="true"
PUT /collections/terraforming/index
{
    "field_name": "color",
    "field_schema": "keyword"
}
```

```json withRunButton="true"
PUT /collections/terraforming/index
{
    "field_name": "humidity",
    "field_schema": {
       "type": "integer",
        "range": true
    }
}
```

## Adım 4: Filtreleme örnekleri

### Tam eşleşme ile filtreleme

Son olarak, bu sorgu, basit bir `match` koşulu kullanarak `color` değeri `"black"` olan noktaları getirir:

```json withRunButton="true"
POST collections/terraforming/points/scroll
{
  "filter": {
    "must": [
      {
        "key": "color",
        "match": {
          "value": "black"
        }
      }
    ]
  },
  "limit": 3,
  "with_payload": true
}
```

### `must` cümlesi ile birleşik filtre

Bu örnekte sorgu, `life` değeri `true` ve `color` değeri `"green"` olan noktaları döndürür. Bir noktanın döndürülmesi için bu must koşullarının her ikisinin de karşılanması gerekir.

```json withRunButton=true
POST collections/terraforming/points/scroll
{
  "filter": {
    "must": [
      { "key": "life", "match": { "value": true } },
      { "key": "color", "match": { "value": "green" } }
    ]
  },
  "limit": 3,
  "with_payload": true
}
```

### `should` cümlesi ile filtreleme

Burada, `life` değeri `false` veya `color` değeri `"black"` olan noktalar arıyorsunuz. Bu koşullar *should* cümleleri olarak çalışır, yani her iki kritere veya bunlardan yalnızca birine uyan noktalar döndürülür:

```json withRunButton=true
POST collections/terraforming/points/scroll
{
  "filter": {
    "should": [
      {
        "key": "life",
        "match": { "value": false }
      }, {
        "key": "color",
        "match": { "value": "black" }
      }
    ]
  }
}
```

### `must_not` cümlesi ile filtreleme

Bu sorgu, `life` değeri `false` olan noktaları filtreler. Bu koşula uyan noktalar sonuçlardan çıkarılır.

```json withRunButton=true
POST collections/terraforming/points/scroll
{
  "filter": {
    "must_not": [
      {
       "key": "life",
       "match": { "value": false }
      }
    ]
  },
  "limit": 3,
  "with_payload": true
}
```

### `range` koşulu ile filtreleme

Bu sorgu, noktaları `humidity` aralığına göre filtreler. Burada `humidity` değeri tam olarak 40 olmalıdır:

```json withRunButton="true"
POST collections/terraforming/points/scroll
{
  "filter": {
    "must": [
      {
       "key": "humidity",
       "range": {
         "gte": 40,
         "lte": 40
       }
      }
    ]
  },
  "limit": 3,
  "with_payload": true
}
```
