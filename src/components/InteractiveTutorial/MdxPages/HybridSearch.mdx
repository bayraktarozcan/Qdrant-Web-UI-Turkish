# Seyrek ve Yoğun Vektörlerle Hibrit Arama

Bu eğitimde, hem seyrek hem de yoğun vektörlere odaklanarak Qdrant'ta hibrit aramayı keşfetmeye devam edeceğiz.
Bu sefer, `terraforming_plans` ile ilgili bir koleksiyonla çalışacağız ve her veri noktasının `payload`'unda içeriğinin kısa bir açıklaması olacak.

## Adım 1: Seyrek vektörlerle koleksiyon oluşturun

`terraforming_plans` adında bir koleksiyon oluşturarak başlayacağız. Bu koleksiyon, anlamsal benzerlik için yoğun vektörleri ve anahtar kelime tabanlı arama için seyrek vektörleri destekleyecek.

```json withRunButton=true
PUT /collections/terraforming_plans
{
    "vectors": {
        "size": 4,  
        "distance": "Cosine"  
    },
    "sparse_vectors": {
        "keywords": { }
    }
}
```

### Açıklama:
- **`vectors`**: Yoğun vektör uzayını 4 boyutla, mesafe ölçümü için cosine similarity kullanarak yapılandırır.
- **`sparse_vectors`**: Koleksiyonu, anahtar kelime tabanlı indeksleme için seyrek vektörleri destekleyecek şekilde yapılandırır.

---

## Adım 2: Açıklamalarla veri noktaları ekleyin

Şimdi, her biri farklı bir gök cismi (Mars, Jupiter ve Venus) ile ilgili olan `terraforming_plans` koleksiyonuna üç veri noktası ekleyeceğiz. Her nokta, `payload`'da bir `description` ile birlikte hem yoğun hem de seyrek bir vektöre sahip olacak.

```json withRunButton=true
PUT /collections/terraforming_plans/points
{
    "points": [
        {
            "id": 1,  
            "vector": {
                "": [0.02, 0.4, 0.5, 0.9],   // Dense vector
                "keywords": {
                   "indices": [1, 42],    // Sparse for "rocky" and "Mars"
                   "values": [0.22, 0.8]  // Weights for these keywords
                }
            },
            "payload": {
                "description": "Plans about Mars colonization."
            }
        },
        {
            "id": 2,  
            "vector": {
                "": [0.3, 0.1, 0.6, 0.4],   
                "keywords": {
                   "indices": [2, 35],    // Sparse for "gas giant" and "icy"
                   "values": [0.15, 0.65]  // Weights for these keywords
                }
            },
            "payload": {
                "description": "Study on Jupiter gas composition."
            }
        },
        {
            "id": 3,  
            "vector": {
                "": [0.7, 0.5, 0.3, 0.8],   
                "keywords": {
                   "indices": [10, 42],    // Sparse for "Venus" and "rocky"
                   "values": [0.3, 0.5]    // Weights for these keywords
                }
            },
            "payload": {
                "description": "Venus geological terrain analysis."
            }
        }
    ]
}
```

### Açıklama:
- **Yoğun vektör**: Veri noktasının anlamsal özelliklerini sayısal biçimde temsil eder.
- **Seyrek vektör (`keywords`)**: Anahtar kelime özelliklerini temsil eder; `indices` belirli anahtar kelimelere, `values` ise bunların alaka düzeyine eşlenir.
- **`payload`**: Her vektörün neyi temsil ettiğini anlamayı kolaylaştıran, veri noktasının içeriği hakkında kısa bir açıklama sağlar.

---

## Adım 3: Hibrit arama gerçekleştirin

Ardından, Karşılıklı Sıra Füzyonu (RRF) kullanarak hem anahtar kelime tabanlı (seyrek) hem de anlamsal (yoğun) aramayı birleştiren `terraforming_plans` koleksiyonunda hibrit bir arama gerçekleştirin.

```json withRunButton=true
POST /collections/terraforming_plans/points/query
{
    "prefetch": [
        {
            "query": { 
                "indices": [1, 42],   
                "values": [0.22, 0.8]  
            },
            "using": "keywords",
            "limit": 20
        },
        {
            "query": [0.01, 0.45, 0.67, 0.89],
            "using": "",
            "limit": 20
        }
    ],
    "query": { "fusion": "rrf" },  // Reciprocal rank fusion
    "limit": 10,
    "with_payload": true
}
```

### Açıklama:
- **`prefetch`**: İki alt sorgu içerir:
  - **Anahtar kelime tabanlı sorgu**: Anahtar kelime alaka düzeyine göre arama yapmak için seyrek vektörü (keywords) kullanır.
  - **Yoğun vektör sorgusu**: Anlamsal benzerlik araması için yoğun vektörü kullanır.
- **`fusion: rrf`**: Her iki sorgudan gelen sonuçları Karşılıklı Sıra Füzyonu (RRF) kullanarak birleştirir ve her iki aramada da yüksek sıralanan noktalara öncelik verir.
- **`limit`**: Sonuçları en iyi 10 ile sınırlar.

---

## Özet

Bu eğitimde:
1. `terraforming_plans` adında, hem yoğun hem de seyrek vektörleri kullanarak hibrit aramayı destekleyen bir Qdrant koleksiyonu oluşturduk.
2. Payload'da açıklamaların yanı sıra hem yoğun hem de seyrek vektörlerle veri noktaları ekledik.
3. Karşılıklı Sıra Füzyonu kullanarak anahtar kelime alaka düzeyini ve yoğun vektör benzerliğini birleştiren bir hibrit arama gerçekleştirdik.

Bu yaklaşım, metinsel ve anlamsal arama yeteneklerini birleştiren etkili bir hibrit aramaya olanak tanır ve karmaşık arama gereksinimleri içeren uygulamalarda oldukça kullanışlı olabilir.
